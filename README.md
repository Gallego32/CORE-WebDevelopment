## Uso de las API

### Open Weather API
- Para el hito de la tercera semana he usado una [API](https://openweathermap.org/) que junto con las opciones de geolocalización del buscador muestra el tiempo en el lugar en que estés en el momento de la conexión.

### GitHub API
- Además he usado la Api de [GitHub](https://docs.github.com/en/free-pro-team) para poner en una tabla todos mis repositorios con sus respectivas estrellas. Dentro de la tabla los nombres de los repositorios serán también enlaces para llegar a estos.
